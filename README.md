* [Meta XR Core SDK (UPM)](https://developers.meta.com/horizon/downloads/package/meta-xr-core-sdk/)
* [Meta XR Audio SDK (UPM)](https://developers.meta.com/horizon/downloads/package/meta-xr-audio-sdk/)
